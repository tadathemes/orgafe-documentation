Main & Mega Menu

Orgafe help create the Main menu and Mega menu in one area only.

At the bottom Content area of Header section, click to the existed menu to edit or Add menu button to add new.

  • Input the name of Menu that you want add into Main menu in the "Menu settings"
  • If you want use the Mega menu layout (multiple columns), must select "Active Mega menu" checkbox of "Mega menu settings"
  • If you don't want content in full container, check to "Dropdown ...." option
  • Select "Mega menu column number". The column from 1 to 6 columns.
  • Input the Mega menu structure:

    This Mega menu support 3 types:

    1. Link list: show the Navigation. Select up to 6 Navigation links column. The shortcut is l
    2. Text: show the custom text. Only 1 column. The shortcut is t
    3. Image: show the navigation. Select up to 4 Images column. The shortcut is i

    Input the column type by shortcut text that you want in each column and separated by :

    Then scroll down to config the column type.

Explain guide for Sidebar option



1. Sidebar - Related Collections

It will display collections listing.

There are 3 method to show collections listing:
  1. Auto get All Collections: it will show all collections in your site with Alphabet order and can not limit or re-oder.
  2. Static Navigation listing: it will show collections from the selected navigation (link list).
  3. Dynamic Navigation listing: it will show collections from a lot of navigation. You can set each collection have each link.

Explain for Dynamic Navigation listing (method #3): must set up navigation (link list) for each collection with syntax: Collection_handle:link_list

Example: you want set collections listing for Fruit collection (handle is fruit-category), you must create the navigation (link list) with links of Fruit first (handle is fruit-1), then input the syntax is fruit-category:fruit-1
the array follow separated by |





2. Advanced Ajax Tags Filter

This is Tags filter function. So, you must tag and config the tags of your products

  1. Tag these tags that you want set in Filter into your products

  2. Group these tags in Customize theme > Section tab > Collection pages > scroll to ADVANCED AJAX TAG FILTER area

Note: it will only show the tag filter in listing products have this value.
